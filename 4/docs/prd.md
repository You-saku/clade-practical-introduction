# プロダクト要求定義書

## 背景
開発者がGitHubのpublicリポジトリ情報を素早く確認したい場面は多い。ブラウザを開かずにターミナルから直接リポジトリの情報を取得できるCLIツールがあれば、開発ワークフローを効率化できる。

## 解決したい課題
- ブラウザを開かずにGitHubリポジトリの基本情報を確認したい
- 複数リポジトリの情報を素早く比較・確認したい
- ユーザーが公開しているリポジトリ一覧を一目で把握したい

## ターゲットユーザー
- 日常的にターミナルで作業する開発者

## 成功基準
- コマンド一つでリポジトリ情報を取得できる
- 認証なしでpublicリポジトリの情報を取得できる
