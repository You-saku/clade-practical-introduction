# 要件: 初期実装

## 概要
GitHub publicリポジトリ情報を取得するCLIアプリ `ghrepo` の初期実装を行う。

## 実装する機能
1. **リポジトリ情報取得 (`repo get <owner>/<repo>`)** — 指定リポジトリの詳細情報を表示
2. **ユーザーリポジトリ一覧 (`user repos <username>`)** — ユーザーのpublicリポジトリ一覧を表示

## 要件詳細

### リポジトリ情報取得
- GitHub REST API v3 (`GET /repos/{owner}/{repo}`) を使用
- 表示項目: リポジトリ名、説明、言語、スター数、フォーク数、作成日、最終更新日、URL
- 存在しないリポジトリの場合はエラーメッセージを表示

### ユーザーリポジトリ一覧
- GitHub REST API v3 (`GET /users/{username}/repos`) を使用
- 表示項目: リポジトリ名、説明、言語、スター数
- `--sort` オプション: stars / updated / name (デフォルト: updated)
- `--limit` オプション: 表示件数 (デフォルト: 10)
- 存在しないユーザーの場合はエラーメッセージを表示

## 非機能要件
- APIタイムアウト: 10秒
- 認証なし (レートリミット: 60リクエスト/時間)
